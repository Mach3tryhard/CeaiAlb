# Proiectare Logica
## Informatii

    Ef - Nota examen sesiune (10 pct maxim)
    Ep - Nota examen partial (10 pct maxim)
    L - Nota Colocviu (10 pct maxim)
    N = 0.35*Ef + 0.35*Ep + 0.3*L;
